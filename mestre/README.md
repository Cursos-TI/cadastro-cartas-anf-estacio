# Super Trunfo dos Países - Mestre

Histórico da versão "Mestre" da tarefa.

## Implementações

- Cálculo do Super Poder, acumulando as características previamente inseridas e calculadas de cada cidade.
- Comparação das características de cada carta e exibição do resultado das comparações.
- Sem estrutura de repetição, structs ou funções auxiliares, conforme orientado.

## Observações

- Foi utilizado um truque algébrico para gerar o número da carta vencedora no output. Isto foi feito para evitar o uso de blocos condicionais, seguindo orientações de passos anteriores.
